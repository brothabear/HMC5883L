Initialize the magnetometer communications.

Starts I2\+C communication with the \hyperlink{class_h_m_c5883_l}{H\+M\+C5883\+L} magnetometer. This creates the Wire interface with the device. If the {\ttfamily no\+Config} parameter is set to {\ttfamily false} (default), the magnetometer is also explicitly initialized with the device default values for the various configuration parameters\+:

\begin{TabularC}{2}
\hline
\rowcolor{lightgray}{\bf Parameter }&{\bf Default value  }\\\cline{1-2}
Gain &{\ttfamily \mbox{[}H\+M\+C\+\_\+\+G\+A\+I\+N130\mbox{]}} 1090 L\+Sb/\+Gauss (+/-\/ 1.\+3 G sensor range) \\\cline{1-2}
Averaging rate &{\ttfamily \mbox{[}H\+M\+C\+\_\+\+A\+V\+G1\mbox{]}} 1 point per sample \\\cline{1-2}
Data output rate &{\ttfamily \mbox{[}H\+M\+C\+\_\+\+R\+A\+T\+E1500\mbox{]}} 15 Hz \\\cline{1-2}
Measurement mode &{\ttfamily \mbox{[}H\+M\+C\+\_\+\+Measurement\+Idle\mbox{]}} Idle mode \\\cline{1-2}
Bias mode &{\ttfamily \mbox{[}H\+M\+C\+\_\+\+B\+I\+A\+S\+\_\+\+N\+O\+N\+E\mbox{]}} No bias \\\cline{1-2}
\end{TabularC}
If {\ttfamily no\+Config} is set to {\ttfamily true}, this will request the values of some of the parameters already set, so as to ensure the accuracy of calls to functions such as {\ttfamily get\+Delay()}, which may use cached values for compass parameters.

\end{DoxyParams}

\end{DoxyReturn}
