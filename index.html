HMC5883L

Library for I2C communication with HMC5883L magnetometers.

Download this project as a .zip file Download this project as a tar.gz file

A library for I2C interaction with HMC5883L 3-axis digital magnetometers. More information about the HMC5883L can be found in the datasheet provided by Honeywell.

The I2CDevice library provides an interface between the HMC5883L class and Arduino microcontrollers, and can be replaced by any equivalent interface to support other interfaces.
